## Описание проекта

Dota 2 Stats Bot — это Telegram-бот, разработанный для быстрого и удобного получения статистики игроков и героев популярной игры Dota 2. Бот позволяет пользователям получать актуальную информацию о своем или чужом игровом профиле, статистику по лучшим героям, а также детальные данные по конкретному персонажу, не прибегая к запуску игры или использованию сторонних веб-сайтов.

Проект демонстрирует использование асинхронного программирования на Python с фреймворком aiogram для взаимодействия с Telegram Bot API и библиотекой requests для работы с внешним OpenDota API.

# Функционал

Бот поддерживает следующие команды:

-   /start: Запускает бота, выводит приветственное сообщение и отображает основную клавиатуру с командами.
-   /profile [ID]: Отображает общую статистику игрока, включая никнейм, оценку MMR, общее количество сыгранных матчей и винрейт.
-   /top [ID]: Показывает трех героев, на которых игрок демонстрирует наилучший винрейт (учитываются герои с минимум 5 сыгранными матчами).
-   /hero [ID] [Имя героя]: Предоставляет детальную статистику по выбранному герою для указанного игрока (количество матчей, винрейт на этом герое).
-   /help: Выводит справочную информацию о доступных командах, тестовых ID и рекомендации по решению распространенных проблем.

## Технологии

Проект разработан с использованием следующих технологий:

-   *Язык программирования:* Python 3.x
-   *Фреймворк для Telegram:* aiogram (асинхронный фреймворк для создания Telegram-ботов)
-   *Библиотеки:*
    -   requests: Для выполнения HTTP-запросов к внешним API (OpenDota API).
    -   asyncio: Встроенная библиотека Python для асинхронного программирования, обеспечивающая эффективную обработку запросов.
-   *API:*
    -   Telegram Bot API: Для взаимодействия с платформой Telegram.
    -   OpenDota API: Для получения игровой статистики Dota 2.

## Решение распространенных проблем

-   Бот сообщает "Профиль не найден или статистика скрыта.":
    Это означает, что у игрока в настройках Dota 2 отключена публичная история матчей. Для активации:
    1.  Зайдите в Dota 2.
    2.  Нажмите на иконку шестеренки (Настройки) в левом верхнем углу.
    3.  Перейдите во вкладку "Сообщество" (Social).
    4.  Установите галочку напротив опции "Общедоступная история матчей" (Expose Public Match Data).
    5.  После изменения настроек может потребоваться 10-15 минут для обновления данных в OpenDota API.
